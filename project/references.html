Sources and Credits

This page documents the content, resources, and external references used for the Restless Minds website.

Original Content

All written content, service descriptions, project explanations, cybersecurity positioning, and website structure were created by Edgar Garcia Morado for the WDD 131 final project.

The website represents Restless Minds as a cybersecurity and digital protection project focused on small businesses, microbusinesses, private communities, practical security services, and educational cybersecurity write-ups.

Images and Visual Assets

The current version of this website does not use external images. The visual design is built with HTML, CSS, typography, cards, icons, and layout styling.

Simple symbols, interface elements, terminal-style components, and service icons are decorative elements created directly in the HTML and CSS for this project.

External Links and Verification Resources

  • INE Security Certification Verification: Used to link to the eJPTv2 certification verification page shown in the About section. View certification verification
  • GitHub: Used to link to the PortHunter project repository in the About section. View PortHunter repository

Cybersecurity Lab Platforms Mentioned

The Write-Ups section references controlled cybersecurity learning platforms where ethical hacking labs and practice machines are commonly completed.

  • DockerLabs: Referenced as a controlled lab platform for cybersecurity practice. Visit DockerLabs
  • TryHackMe: Referenced as a cybersecurity learning platform for guided rooms and practical exercises. Visit TryHackMe
  • Hack The Box: Referenced as a hands-on cybersecurity training platform. Visit Hack The Box
  • VulNyx: Referenced as a cybersecurity lab platform for vulnerable machines and practice environments. Visit VulNyx

Technologies Used

This website was built using HTML, CSS, and JavaScript. JavaScript is used for the mobile navigation, footer dates, dynamic lab filtering, lab progress tracking, testimonials, form handling, and local storage.

No third-party templates, site builders, or drag-and-drop website tools were used.

Academic Project Note

Restless Minds was developed as a course project for WDD 131: Dynamic Web Fundamentals. The site is designed as a professional-style cybersecurity website while meeting the technical requirements of the course project.
